Codex-Bookmarklet
=================

A bookmarklet for beautifying Wordpress Codex

### Installation: 

1. Create a bookmark
2. Copy and paste the code from [codex.min.js](https://github.com/kostasx/Codex-Bookmarklet/blob/master/codex.min.js) into your bookmark.
